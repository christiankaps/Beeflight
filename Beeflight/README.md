# Beeflight
BeeFlight iOS App

## Description
This iOS app shows basic GPS and Sensor data on a dashboard.

Sensor Data:
- GPS Position
- Speed (kph)
- Altitude (m)
- Climbing Speed (m/s)
- Heading (Compass)
- Course (Ground Track)

Languages:
- English
- German

Appearance:
- App Name: Beeflight
- App icon: a bee
- Only portrait mode supported
- App supports light and dark mode based on the system settings
-

Connectivity:
- App shall work independent of an internet connection


Frameworks:
- Swift UI
